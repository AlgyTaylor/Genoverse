What is it?

Genoverse is a portable, customizable, back-end independent genome browser powered by Javascript & HTML5. It allows users to explore data interactively by scrolling and zooming without page reload.

What's so good about it?

Apart from looking cool and being interactive, it can easily be installed to any website and connected to your data source with a few lines of javascript, just like google maps or other web widget. Code for this example:

<!DOCTYPE HTML>
    <html>
      <head>

        <script type="text/javascript" src="js/combined.js">
          // Genoverse configuration
          {
            container      : '#genoverse', // CSS/jQuery style for where to inject genoverse
            width          : 756,
            chromosomeSize : 249250621,    // chromosome 1, human
            // Default location
            chr            : 1,
            start          : 45467084,
            end            : 45473455,

            plugins        : [
              'controlPanel',
              'trackControls',
              'resizer',
              'tooltips'
            ],

            // Tracks configuration
            tracks         : [
              {
                type : 'Scalebar',
                height : 25,
              },

              {
                type : 'Sequence',
                // Sequence source URL, with placeholders for chromosome, start and end positions, which would be automatically replaced for every AJAX request
                url  : 'http://www.ensembl.org/das/Homo_sapiens.GRCh37.reference/sequence?segment=__CHR__:__START__,__END__', 

              },          

              {
                name    : 'Transcript',
                type    : 'DAS.Transcript',
                // For DAS tracks we use 'source' rather than just 'url', DAS tracks will workout url for the data and stylesheet
                source  : 'http://www.ensembl.org/das/Homo_sapiens.GRCh37.transcript',
                filter  : {
                  type  : [ 'transcript', 'translation' ]
                },
                // populateMenu invoked once user clicks on any feature of the track
                // -> feature that user clicks on (JSON object)
                // <- json for the menu items
                populateMenu: function (feature) {
                  return {
                    title: feature.label,
                    start: feature.start,
                    stop : feature.end,
                    Links: '<a href="#">e!</a>'
                  };
                }
              },

            ]
          }

        </script>


      </head>
      <body>

      <div id="genoverse"></div>

      </body>
    </html>

How does it work?

Data is loaded dynamically via AJAX and visualized inside your web browser using HTML5 CANVAS element

Genoverse works with a variety of formats, such as XML, JSON or delimited text files, and can be customized to parse and display any data source as required.
